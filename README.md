# Minecraft Bitnary Server
Minecraft Server used by Bitnary Tecnologia™ and UnioIT Software House™ in Nov/2019.

## Requirements
Java Runtime Environment:
- Node.js (v12.22.9)
```sh
sudo apt install default-jre
```

## Running
Start server (Default port 25565):
```sh
./start.sh
```

Stop server:
```sh
./stop.sh
```
